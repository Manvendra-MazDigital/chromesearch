Essentially, Chrome Apps are a way to use HTML5 to develop real, native-feeling applications. Applications that look and feel like native ones. They run outside the browser. They work offline by default. They work on desktop as well as mobile. So it's a great way for developers to build real native experiences.

It's local on your machine like an installed app is. And you can run the app when you're not connected, or when you have a connection that's not so good, and it just behaves normally.
